Browser Game

Group Project

This project is a browser game inspired by the WarioWare titles.
The player selects the desired game mode and tries to beat as many minigames as they can.
Minigames increase in difficulty as the player's score increases.
Once the game is over, players can submit their score to a leaderboard that resets daily.
The project uses Node.js to handle web server interactions.
