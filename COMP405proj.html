SQL Database Viewer

Group Project

This program is used to view a Pokemon SQL database created by scraping data from various Pokemon wiki pages.
The scraper builds the database by searching source HTML with a regex for each data.
Each table is created automatically, so once the scraper is done it can be browsed using the accompanying GUI program.
The GUI allows custom SQL execution as well as premade sorting options.
